Workspace - NodeMCU (ESP8266)

Install Latest Firmware on your NodeMCU or ESP8266

  1. NodeMCU comes with the Lua firmware pre-loaded. ESP8266's don't have Lua preloaded. To load the latest firmware version you should build your own at NodeMCU-build.com or you can download a tested working version from this workspace at nodemcu-firmware-2016-01-10.bin
  2. You can flash the firmware by using nodemcu-flasher on Windows or Python Esptool on Linux. You can also try the flasher tools from Espressif Flash Download Tools page.
